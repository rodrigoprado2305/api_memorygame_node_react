### 🛠️ **Detail Stack:**
- Frontend: React, Vite
- Backend: Node.js, Express
- Database: MongoDB

### 🚀 **Setup Guide:**
1. Set up the backend:
   ```
   cd ./backend
   npm install
   npm start
   ```
2. Set up the frontend:
   ```
   cd ./frontend
   npm install
   npm run dev
   ```
